CSS Needs

Brief History

Css was created by Håkon Wium Lie who was working with Tim Berners-Lee, who was developing HTML at the time. Css is used for styling HTML webpages and used by almost all HTML webpages.

Basic Needs

Css files need to be linked to with, link rel="stylesheet" href="INSERT.CSS File", Of course the link will be in the HTML file and inside the Css file you won't be specifying the DOCTYPE, you will only need to give it the .css file extension. Oh and don't forget to enclose it with <>!
