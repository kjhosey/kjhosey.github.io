What is HTML?

HTML stands for Hypertext Markup Language and is what most websites are coded in. Tim Berners-Lee was the person who first created HTML back in 1993, but nowadays HTML is maintained by WHATWG (Web Hypertext Application Technology Working Group).

There are different versions of HTML, below is a list of HTML versions and release.

HTML Ver. Release
HTML 1991
HTML+ 1993
HTML 2.0 1995
HTML 3.2 1997
HTML 4.01 1999
XHTML 1.0 2000
HTML 5 2012
XHTML 2013
